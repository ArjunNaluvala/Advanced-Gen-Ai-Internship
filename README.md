# Gen AI Internship Entrance Test – Python & EDA

This repository contains the solutions for the **Gen AI Internship Entrance Test**.  
The tasks were based on **Python programming and Exploratory Data Analysis (EDA)** using Numpy,Pandas.

## Objective
The purpose of this test was to evaluate problem-solving skills, Python logic building, and basic data analysis using real datasets.

## Tools Used
- Python
- Pandas
- NumPy

## Tasks
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Filtering and analyzing dataset using Pandas

## Files
- `eda_analysis.ipynb` – EDA implementation
- `dataset.csv` – Dataset used for analysis
- `python_tasks.py` – Python logic based tasks
